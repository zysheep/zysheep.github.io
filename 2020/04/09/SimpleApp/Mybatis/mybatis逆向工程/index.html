报错:The specified target project directory src/main/java/ does not exis
原因:逆向工程的targetProject是相对于主工程的目录,如果直接在项目下使用不用加模块名,即 targetProject=”src\main\java”就可以生成代码,如果在子模块工程中使用前面需要加字模块名,即targetProject=”子模块名\src\main\java”我这里是直接在项目下生成的,所以你不用加子目录
